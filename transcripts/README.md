# AI session transcripts

Save your exported AI session transcripts here (Codex / Claude Code).

These are the evidence for the **Innovation & AI collaboration** criterion, so export them as you go rather than leaving it to the end. The onboarding pack covers how to export from each tool.

## Files

- `setup-session-2026-06-24.md` - setup transcript and verification notes.
- `prompt-history-2026-06-24.md` - prompt-only history for the same session.
- `full-session-transcript-2026-06-24.md` - consolidated session transcript
  with user prompts, assistant actions, checks, and final submission notes.
